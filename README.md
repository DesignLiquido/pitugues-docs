# pitugues-docs

Documentação da linguagem Pituguês. "Python" em português é "Píton". Como Pituguês junta elementos da linguagem Python com português, criamos o nome "pituguês". 

[A documentação geral vive na Wiki deste repositório](https://github.com/DesignLiquido/pitugues-docs/wiki), sendo o restante do repositório dedicado a arquivos e diretórios de apoio.
